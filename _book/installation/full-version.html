Full Version

Full Version

Run the following commands on your terminal to setup React Native Food Ordering App on your system.

Installation

  • Opt #1. Download ZIP

Not familiar with Git? Click here to download the Full Version of the theme. Extract the contents of ZIP file after downloading. Downloading ZIP file does not help you to sync with further updates of Food Ordering App.

  • Opt #2. Clone using GitStrap Web Client

To setup the Full Version for Food Ordering App on your system, with gitstrap tools to sync your app with constant updates, clone the repo. Click here to clone the theme using GitStrap.

  • Install packages for Full Version

    cd SwiggyApp
npm install
react-native link

  • To simulate for iOS

    • Method One:
      • Open the project in XCode from ios/SwiggyApp.xcworkspace.
      • Hit Build button to simulate.
    • Method Two:
      • Type react-native run-ios in your terminal.

  • To simulate for Android

    • Make sure you have an Android emulator installed and running.
    • Type react-native run-android in your terminal.
